# Cpp dice

A simple C++ console application that simulates rolling a six-sided die.

## Features

- Random dice roll
- Easy to understand code
- Great for C++ beginners

## How to run

1. Open the project in Visual Studio.
2. Build and run.
3. See what number you got
4. Type y if you want to continue and n if you want to stop the program (you can type Y and N too)
